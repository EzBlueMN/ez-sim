# EZ-Sim: Proto-Gouvernement AI

## Objectif
EZ-Sim est un prototype expérimental de “proto-gouvernement AI”, visant à simuler :
- une population d’agents (humains simulés),
- une économie simplifiée (monnaie, transactions),
- des mécanismes de gouvernance (vote, règles exécutables),
- un suivi visuel des décisions et allocations.

L'objectif est de créer un système reproductible, testable et ouvert, permettant d’explorer des alternatives aux structures de gouvernance classiques.

## Structure du projet
- `src/simulation/` : agents, environnement, interactions
- `src/governance/` : règles de décision, vote, gouvernance
- `src/economy/` : monnaie, transactions, redistribution
- `docs/` : documentation, architecture
- `tests/` : tests unitaires
